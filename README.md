## 환경 설정
1. Python = 3.10.15
2. Git으로 Repository 클론
```shell
git clone https://github.com/ha789ha/smart_watch.git
cd smart_watch
```
3. Python Requirements 설치
```shell
pip install -r requirements
```
4. config.py
```shell
access_token = "your google cloud api refresh token"
```  
4. app.py 실행
```shell
python app.py
```
